# Laravel-learn

## Laravel

### [Install Laravel](https://github.com/datlt4/PHP-learn/blob/main/README.md#install-laravel)

### Create project

```bash
composer create-project laravel/laravel Laravel-learn
```

- Reinstall dependencies

```bash
composer update
composer install
```
